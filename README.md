# Progetto jbudget

Questo repository contiene il codice sorgente per `jbudget`, un'applicazione desktop per la gestione del budget personale costruita con Java e JavaFX

## Avvio Rapido (Riga di Comando)

1.  **Clona il repository:**

2.  **Costruisci il progetto (Build):**
    Questo comando compilerà il codice sorgente ed eseguirà i test.
    ```cmd
    .\gradlew.bat build
    ```

3.  **Esegui l'applicazione (Run):**
    Una volta terminata la build, puoi avviare l'applicazione:
    ```cmd
    .\gradlew.bat run
    ```

## Esecuzione in un IDE (Consigliato)

Puoi importare questo progetto nel tuo IDE preferito (come IntelliJ IDEA o Eclipse).

1.  Apri il tuo IDE e seleziona "Apri" o "Importa".
2.  Seleziona la cartella principale del progetto (quella che contiene il file `build.gradle`).
3.  L'IDE dovrebbe riconoscere automaticamente il progetto Gradle.

## Tecnologie Utilizzate

* **Java**
* **Gradle**
* **JavaFX**
* **Hibernate / Jakarta Persistence**
* **H2 Database**
* **Lombok**
